# electriz

A Python chess project using the python-chess library.

## Description

This project implements chess functionality using the python-chess library. It includes basic chess board operations and random move generation.

## Installation

```bash
pip install python-chess
```

## Usage

Run the main script:

```bash
python electriz.py
```

## Project Structure

```
electriz/
├── data/          # Data files
├── src/           # Source code
├── tests/         # Test files
└── electriz.py    # Main script
```

## Dependencies

- python-chess
